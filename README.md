# T3 App with experimental app dir

This project was built using the T3 stack but modified to the new Next.js app directory.

## Contains the following

- [Next.js](https://nextjs.org)
- [NextAuth.js](https://next-auth.js.org)
- [Prisma](https://prisma.io)
- [Tailwind CSS](https://tailwindcss.com)
- [tRPC](https://trpc.io)
